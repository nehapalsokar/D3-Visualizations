3 easy steps:

1.Default functionality:
*Poverty rate values over 20 are highlighted by hovering mouse.
*Clicking on "Remove poverty-rate values less than entered value" button deletes values below 20
2.Enter any poverty rate value between 5-25 and click "Go!" button to change default value of 20.
*Hover mouse over values above new entered value to see change in color
*Now clicking on "Remove poverty-rate values less than entered value" button deletes values below new entered value
3.Try changing the background color by clicking the button!
